Benvenuto su FileDeck

Prima di iniziare, leggi come funziona. FileDeck agisce sui file veri del tuo disco, quindi è bene saperlo.

⚡ Operazioni reali. Quando sposti o cancelli, FileDeck modifica davvero i file sul tuo computer — non genera un report o una simulazione.
! Niente Cestino, niente annulla. I file cancellati spariscono subito; lo spostamento copia nella destinazione e poi rimuove l'originale. Controlla sempre la selezione prima di confermare.
🔒 Privacy totale. Tutto avviene nel tuo browser: nessun file viene caricato online, nessun account, nessun server. I tuoi dati non lasciano il computer.
🖥️ Solo browser desktop compatibili. Servono Chrome, Edge o Opera su computer. Su Safari, Firefox e iPhone/Android l'accesso ai file non è disponibile.

Usi FileDeck sotto la tua responsabilità: verifica i file selezionati e, se possibile, tieni un backup di ciò che conta. Software fornito “così com'è”, senza garanzie (licenza MIT).
